# `final_activities.ipynb`

This notebook contains the full activity flow from start to end:

- `Activity 1`: text preprocessing, cleaning, tokenisation, emoticon-aware tokenisation, and a complete normalisation pipeline.
- `Activity 2`: applying the preprocessing pipeline to `reviews.csv`, creating tokens, and analysing dataset-level token statistics.
- `Activity 3`: building a vocabulary, vectorising text with Bag of Words, and creating the document-term matrix.
- `Activity 4`: computing TF, IDF, and TF-IDF, then comparing the results with `scikit-learn`.
- Written discussion and reflection answers are included alongside the coding parts.
