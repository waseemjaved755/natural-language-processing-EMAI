# Natural Language Processing for EMAI(Emotional AI)

- Text preprocessing, cleaning, tokenisation, emoticon-aware tokenisation, and a complete normalisation pipeline.
- Applying the preprocessing pipeline to `reviews.csv`, creating tokens, and analysing dataset-level token statistics.
- Building a vocabulary, vectorising text with Bag of Words, and creating the document-term matrix.
- Computing TF, IDF, and TF-IDF, then comparing the results with `scikit-learn`.
- Written discussion and reflection answers are included alongside the coding parts.
